# Custom Styling Module for Odoo Login Screen

This module customizes the styling of the Odoo login screen with **Cymax Technologies' color grading** and design aesthetics.

---

## Features:
- Applies a custom theme to the login page.
- Seamless integration with Cymax Technologies' branding.

---

## Installation Steps:
1. Place this module folder into your **custom addons** directory.
   - If you have a separate directory for custom addons, add it there.
2. Update the module list:
   - Navigate to **Apps** in your Odoo instance.
   - Click on **Update App List** to refresh the module list.
3. Search for **Custom Login Page** in the Apps menu.
4. Install the module to activate the custom login page styling.

---

## Notes:
- Once activated, the custom login page styling will automatically apply to your Odoo instance.
- Ensure the **custom addons** directory is included in the `addons_path` of your Odoo configuration file (`odoo.conf`).
